# Contributing to Algo-Visualizer Core

First off, thank you for considering contributing to this project! It's people like you that make the open-source community such an amazing place to learn, inspire, and create.

## How Can I Contribute?

### Reporting Bugs
If you find a bug or unexpected behavior, please open an **Issue** and include:
- A clear and descriptive title.
- Steps to reproduce the problem.
- Your environment (OS, Browser, Python version).

### Suggesting Enhancements
I am always looking for ways to improve this engine! If you have ideas for new features (like adding Quick Sort or Merge Sort), feel free to:
1. Open an Issue with the tag `enhancement`.
2. Describe the feature and why it would be useful.

### Pull Requests
1. **Fork** the repository.
2. Create your **Feature Branch** (`git checkout -b feature/AmazingFeature`).
3. **Commit** your changes (`git commit -m 'Add some AmazingFeature'`).
4. **Push** to the branch (`git push origin feature/AmazingFeature`).
5. Open a **Pull Request**.

## Style Guidelines
- Maintain the **v6.6.6** visual aesthetic.
- Ensure all new logic is **Asynchronous** (for smooth animations).
- Keep the code clean and well-commented in English.
