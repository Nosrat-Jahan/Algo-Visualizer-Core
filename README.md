# 📊 Algo-Visualizer Core (v6.6.6)

**Algo-Visualizer Core** is a sophisticated web-based sorting simulator engineered to demonstrate the mechanics of sorting algorithms in real-time. Built as part of a **BSc in CSE** portfolio, this platform provides a hands-on experience in understanding **Bubble Sort** through dynamic visual mapping and state transitions.

---

## 🚀 Key Features
- **Real-Time Simulation**: Watch the Bubble Sort algorithm in action with smooth bar animations.
- **Dual Mode UI**: Seamlessly switch between **Dark Mode** and **Light-Gray Mode** for optimal accessibility.
- **Dynamic Refresh**: Generate new random datasets with a single click to test different sorting scenarios.
- **Async Logic**: Powered by asynchronous JavaScript to ensure fluid visual updates without blocking the main thread.

## 🛠️ Tech Stack
- **Backend:** Python 3.x / Flask Framework
- **Frontend:** HTML5, CSS3 (Modern Variables), JavaScript (ES6+)
- **Architecture:** Client-Server model for lightweight local hosting.
